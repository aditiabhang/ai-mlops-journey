## 🎯 The Goal

Go from zero to **AWS Certified AI Practitioner (AIF-C01)** — with real projects, not just flashcards.

Along the way: build containers, deploy on K8s, call LLMs, wire up RAG pipelines, and ship a capstone AI agent. Every sprint ends with something you can demo.
